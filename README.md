微众银行多方大数据隐私计算平台[WeDPR](https://github.com/WeBankBlockchain/WeDPR)核心组件库，包括：

- 丰富的隐私计算算法组件，支持隐私求交集、匿踪查询、多方联合分析(隐私SQL)、数据预处理、隐私建模和预测等能力，以满足多样化的业务场景
- 统一网关：稳定的跨机构网络通信组件，支持基于Rip协议的路由转发协议
